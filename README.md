# My nushell dotfiles

- [oh-my-posh](https://ohmyposh.dev/)
- [gorss](https://github.com/Lallassu/gorss)
- [wiki-tui](https://github.com/Builditluc/wiki-tui)

- local.path -- path list for local host. path per line
